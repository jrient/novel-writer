## 第一部分：通用规律

基于对45部剧本评审档案的深度结构化分析，以下是为您提炼的剧本评估7大维度核心规律与实操指南：

### 1. premise_innovation（题材与设定创新度）

**核心规律**
* 创新是双刃剑，高概念设定必须匹配相应的执行力，否则极易产生割裂感。
* 纯粹的设定噱头若无实质剧情支撑，无法掩盖套路化的本质。
* 常规题材若能在商业属性和爽点上达标，即便创新度低依然有签约机会。
* 奇葩或小众设定存在认知门槛，需谨慎评估受众接受度。

**正例（签）**
* **《天降魔丸，纨绔爹爹被逼成栋梁！》**（签，创新度8）：设定极具新意且执行到位，各维度均无短板，直接过会。
* **《海王猎诡：我把替身船玩成了后宫》**（签，创新度8）：将猎诡与后宫结合，设定新颖且情绪调动直接，获得责编高度认可。

**反例（拒）**
* **《高武：我的战力是银行卡余额》**（拒，创新度7/8）：核心设定极具看点，但剧本整体写法和开局桥段过于古早老套，“旧瓶装新酒”导致拒稿。
* **《满级刺绣一卡》**（拒，创新度3）：刺绣元素仅停留在噱头层面，本质依然是套路化的普通言情，被判定为缺乏创新。

**量化锚点**
* 创新度 ≥8 且无明显逻辑硬伤的剧本，过会率极高；但若创新度 ≤3 且为原创剧本，极易因“同质化”被毙。

**可执行建议**
* **DO**：为高概念设定寻找最匹配的叙事基调（如奇葩设定更适合走幽默轻喜剧路线，而非严肃苦大仇深）。
* **DON'T**：不要把新奇职业或系统仅仅当成一层“皮”，核心剧情却照搬十年前的老套路。

**核心规律**

## 第二部分：类型专项

### 原创 / 萌宝

基于提供的4部“原创 / 萌宝”剧本评审档案，以下是该类型的独特要求及规律总结：

### 1. 这个类型特别看重什么

*   **极致的人设反差与反套路创新**
    萌宝赛道极度内卷，常规的“乖巧/天才/算命”设定已无法满足市场，必须打破刻板印象。
    *   *案例*：《天降魔丸》赋予萌宝“好赌、海量、武力值高”的反差属性，上演“魔法打败魔法”的反向带娃（萌宝降维打击纨绔爹爹），凭借强烈的喜剧反差成功过稿（创新分高达8分）。
*   **极快的开局节奏与核心人物宿命绑定**
    开局必须迅速建立萌宝与父母的核心关系，不能慢热。
    *   *案例*：《天降魔丸》开局利用“皇帝求女”直接让萌宝从天而降砸中男主，瞬间完成宿命绑定，节奏极快；反之，《诛仙台上捡个爹》因男主第五集才出场，被多位责编批评“出场太晚”、“不符合短剧节奏”。
*   **父母（特别是女主）的独立高光与主视角**
    萌宝虽然是核心卖点，但不能沦为纯粹的“萌宝秀”，父母必须有自己的高光时刻。
    *   *案例*：《诛仙台上捡个爹》中，责编明确指出“希望之后女主高光多一些”、“主视角改为女主”，强调不能让女主仅仅沦为萌宝助攻的客体。
*   **爽点/虐点的逻辑自洽**
    无论是走虐恋风还是爽文风，情绪的来源必须符合基本常识，不能为了情绪牺牲逻辑。
    *   *案例*：《妈妈，这次换我爱你》试图营造“萌宝护父”的极致虐心感，但设定“男主腿残疾不影响打台球，却沦落到靠女儿捡废品”存在严重逻辑硬伤，导致剧本被拒。

### 2. 这个类型的常见翻车点

*   **元素无脑堆砌，陷入严重同质化**
    以为集齐热门元素就能爆，但缺乏核心差异化，导致剧情重复度极高。
    *   *案例*：《诛仙台上捡个爹》虽然集齐了“马甲、奶团子、双强、带球跑”等下沉市场热门元素，但仍被批评“重复度太高”、“这种很土的萌宝文市面上太多”。《萌宝8岁》也因“几乎是同质化内容”被拒。
*   **开局仅展示萌宝“技能”，缺乏强冲突钩子**
    误以为萌宝展示特长就能留住观众，忽略了剧情张力。
    *   *案例*：《萌宝8岁》第一集仅表现萌宝会算命，被责编一针见血指出“卡点差点意思”，缺乏实质性的剧情钩子。
*   **反派极度降智，冲突推进靠“硬拖”**
    为了凸显萌宝或主角，强行把反派写成只会无脑叫嚣的工具人，导致节奏拖沓。
    *   *案例*：《妈妈，这次换我爱你》中，反派台词极度脸谱化（“在江城我就是王法”），且连续多集在医院走廊上演单方面辱骂和扇耳光，被指极度拖沓刻意。
*   **专业元素叠加生硬，缺乏常识**
    给父母叠加职业属性时不做功课，导致露怯出戏。
    *   *案例*：《妈妈，这次换我爱你》叠加了台球体育事业，但被责编吐槽“台球知识好浅薄”，差异化仅停留在元素的生硬叠加。

### 3. 和其他类型的关键差异

*   **“工具人”属性的倒置**：在常规言情/战神剧中，配角是主角的工具人；但在萌宝剧中，**萌宝是破局和推动剧情的“超级工具”**（如《天降魔丸》中萌宝用仙力解决凡间难题，《诛仙台》中的萌宝助攻），但**情感落脚点必须回归父母**。如果父母沦为萌宝的挂件，剧本就会失去长线吸引力。
*   **对“微创新”的容错率极低**：由于萌宝题材市场饱和度极高（档案中多次出现“同质化”、“重复度高”的评价），常规的“下山+找爹”或“带球跑+虐恋”已经失效。该类型比其他类型更依赖**设定的颠覆性**（如《天降魔丸》的纨绔爹爹被逼成栋梁）才能突围。

### 4. 评分修正建议

在评审“原创 / 萌宝”类型时，建议对以下维度进行权重调整：

*   **加权维度：`premise_innovation` (设定创新度)**
    *   *理由*：赛道极度拥挤，没有反常规设定的萌宝文几乎必死。只要萌宝人设或带娃模式有亮眼反差（如“魔丸降维打击”），应大幅提高此项评分。
*   **加权维度：`opening_hook` (开局钩子) & `pacing_conflict` (节奏与冲突)**
    *   *理由*：萌宝文极易在开局陷入“慢热找人”或“单纯秀技能”的误区。核心人物（爹/妈）出场是否够快、萌宝与父母的绑定是否迅速、开局是否有强冲突，是决定生死的核心指标。
*   **降权维度：单纯的元素叠加 (在评估 `benchmark_differentiation` 时)**
    *   *理由*：不能因为剧本包含了“马甲+双强+带球跑+算命”就给高分。如果这些元素只是生硬缝合，没有产生新的化学反应（如《诛仙台》），其差异化评分应严格打低。
*   **一票否决项（Red Flag 强扣分）：逻辑硬伤与降智**
    *   *理由*：萌宝文受众虽然下沉，但对“为了虐而强行制造不合理的惨”（如《妈妈》中能打台球却只能捡垃圾）或“反派无脑复读机”极度反感。一旦出现此类严重违背常识的设定，`character_depth` 和 `writing_dialogue` 应直接给低分。

### 改编 / 女频

基于提供的12部“改编 / 女频”剧本评审档案，以下是该类型区别于通用规律的独特要求和评审洞察：

### 1. 这个类型特别看重什么？

*   **极致且迅速的“爽点”兑现（打脸反杀不拖泥带水）**
    女频受众对受委屈的容忍度极低，面对挑衅必须有直接、甚至暴力的反击。
    *   *案例*：《八零福星俏娇妻》中女主面对造黄谣的反派，直接采取“物理（窝心脚/棒槌抽打）+言语”双重打击，被责编评价为“挑不出毛病的好看”；《一世容安》中连环解决庶妹、庶姐、继母，爽点密集支撑起了整体精彩度。
*   **符合现代女性价值观的“清醒大女主”人设（反古早小白花）**
    即使是改编古早网文，女主的行为逻辑也必须符合当下独立女性的期待，不能为了制造冲突而降智或展现出软弱。
    *   *案例*：《日久成瘾》中女主面对男主给的500万支票依然去兼职，被责编明确指出“不为金钱折腰的小白花人设过于古早”，需变更动机；《帝国老公狠狠爱》中女主“为了给总裁分担财务压力去酒吧兼职”也被认为脱离现实、不合时宜。
*   **男女主早期的CP互动与“宿命感”**
    即使是主打搞事业或宅斗的女频剧，感情线的前期铺垫也至关重要，男主不能沦为背景板。
    *   *案例*：《二嫁权宦》因男女主前期对手戏太少、缺乏宿命感而遭到多位责编差评；《一世容安》虽被签下，但责编特意警告“男主前10集存在感太低，需要往里塞戏份”。
*   **高度同质化赛道中的“微创新”或“反差感”**
    在穿越、重生、冷宫等红海题材中，引入其他频道的元素或特殊人设能有效破局。
    *   *案例*：《高考750分断绝关系》将男频常见的“断绝关系流”与女频学霸设定结合，并加入女性互助元素，让责编惊呼“这竟然是女频文？还蛮新颖的”。

### 2. 这个类型的常见翻车点

*   **古言语境下滥用现代网络热词（严重破坏沉浸感）**
    穿越剧虽自带现代思维，但台词过度网络化会产生强烈的“小白感”和出戏感。
    *   *案例*：多部《我在冷宫忙种田》的改编稿件因台词出现“老娘”、“一分钟”、“油腻男”、“现场直播”、“视觉污染”等词汇被拒。
*   **反派全员降智、脸谱化（缺乏智商博弈）**
    为了衬托女主的爽点，强行把反派塑造成只会大呼小叫的纸片人，导致剧情像“过家家”。
    *   *案例*：《帝少的重生毒妻》中直接设定反派为“智商不高、嚣张跋扈”；《高考750分断绝关系》中反派为了博同情强行自残，被责编批评“有点虚浮”。
*   **金手指过于万能，剥夺了主角的生存压力**
    过度依赖系统或空间解决一切问题，会导致剧情失去实质性冲突，沦为平淡的日常流水账。
    *   *案例*：《我在冷宫忙种田》中女主靠随身空间轻易获得物资和制冰，缺乏主动克服困难的成就感，导致剧情偏向平淡。
*   **滥用OS（内心独白）交代背景，拖慢节奏**
    女频重生文常犯的错误，试图用内心独白来强化仇恨，反而造成听觉疲劳。
    *   *案例*：《高考750分断绝关系》中女主几乎每个场景都在OS“上一世就是这样……”，被责编吐槽严重拖慢叙事节奏。
*   **在爽点高潮处强行打断情绪**
    *   *案例*：《高考750分断绝关系》在女主自证清白、即将迎来打脸高潮时，安排女主“气晕倒下”，严重阻断了观众的情绪释放，导致爽感大打折扣。

### 3. 和其他类型的关键差异

*   **对“套路”的极高包容度 vs 对“执行力”的严苛要求**
    与其他类型追求新奇设定不同，女频改编剧**不怕老套，就怕不爽**。只要打脸节奏快、爽点密集，即使是“重生+复仇+双强”的纯套路（如《嫡女贵凰》被评为“平平无奇大女主”、“锦绣2.0版本”），依然能拿到80分高分并签约。
*   **男主设定的“双重标准”**
    在男频剧中女主可以是纯挂件，但在女频剧中，男主既要是女主复仇/搞事业的“最强辅助（如权臣、霸总）”，又必须具备极致的情感张力（如《帝国老公狠狠爱》中被认可的“年上大叔感”、《二嫁权宦》中要求的“霸气”）。

### 4. 评分修正建议（针对 改编 / 女频）

*   **加权维度：Payoff Satisfaction（爽点兑现与满足感）**
    *   *理由*：这是女频商业剧的生死线。只要反击果断、打脸密集（如《八零福星俏娇妻》的暴打二流子、《一世容安》的连环反杀），即使设定老套也应给予高分。
*   **加权维度：Character Depth（角色深度，特指女主的现代意识与CP张力）**
    *   *理由*：女频受众极度看重代入感。女主是否具备“清醒独立”特质（拒接小白花）、男女主之间是否有“宿命感”，直接决定剧本的留存率。
*   **降权维度：Premise Innovation（设定创新度）**
    *   *理由*：女频赛道（如重生宅斗、穿书年代文）本身已高度成熟，强求颠覆性创新不切实际。只要能在经典框架内做到“微创新”（如人设反差），设定维度的平庸不应成为拒稿的决定性因素。
*   **降权维度：Writing Dialogue（文笔与台词，仅限华丽度）**
    *   *理由*：不需要多高级的文学性，只要做到“干脆利落、不拖泥带水”（如“对垃圾回收不感兴趣”）即可。但**必须设立一票否决的红线**：一旦古言中出现大量现代网络烂梗（如“金主爸爸”），应直接在台词维度打低分。

### 原创 / 男频

基于提供的12部剧本评审档案，针对“原创 / 男频”类型的通用规律与独特要求，总结如下：

### 1. 这个类型特别看重什么？

男频短剧高度依赖**“新奇设定+极致释放”**的组合，受众对视觉刺激、升级机制和情绪发泄的直接性要求极高。

*   **直观且极致的爽点兑现（拒绝拖泥带水）**
    *   *案例*：《海王猎诡》中主角面对怪物不打嘴炮，直接“物理开颅”并收服女诡；《出租狂飙》中核心爽点通过“出租车飞跃顶级赛车并在空中比中指”这种极具视觉冲击力的方式兑现。男频的爽点必须给得直接、暴力或极具画面感。
*   **设定的微创新与“网感”脑洞**
    *   *案例*：《高武》将战力与金钱直接挂钩（1元=1战力），《姐，咱家南瓜成精了！》将传统的种田流替换为“植物大战僵尸”式的奇幻设定（吃苦瓜放屁、黑芝麻瞬间生发）。这种简单粗暴、自带网梗的设定能迅速抓住男性受众。
*   **快节奏与高频的地图/事件推进**
    *   *案例*：《海王猎诡》前10集就完成了“反杀女诡-秒杀接引使-屠戮水猴群-击杀龟丞相”的密集升级与地图转换，绝不在单一场景过多停留。
*   **带有“颠”感或轻喜剧的反套路基调**
    *   *案例*：《姐，咱家南瓜成精了！》因其强烈的二次元喜剧色彩和“好颠”的特质备受好评；而《高武》虽然设定好，但因为没走轻喜剧路线反而被批评“不适合情绪”。

### 2. 这个类型的常见翻车点

男频作者极易陷入古早网文的路径依赖，或者在追求爽感时忽视了基本的逻辑与受众心理。

*   **开局套路极度老化（“三板斧”失效）**
    *   *案例*：《高武》开局采用“测战力废柴+校花当众背叛+富二代踩脸”；《被杀就满级》采用“底层当牛马抱怨+觉醒系统”。这些十年前的网文套路在当前短剧市场已被多位责编批评为“太典了”、“写的挺老”、“缺乏吸引力”。
*   **憋屈时间过长，爽点严重后置**
    *   *案例*：《被抢车位后，我亮出总裁身份》用整整8集的篇幅去拉扯一个车位纠纷，主角一直在走合法合规路线收集证据，导致责编评价“看点不足，期待一般”；《铁拳镇山河》主角长期处于隐忍和旁观状态，单场景（聚仙楼）占据极大篇幅，节奏拖沓。
*   **受众定位与基调严重错位（毒点）**
    *   *案例*：《我和魅魔大叔刷剧情的日子》作为男频剧，却设定“男魂女身”去攻略其他男性，导致责编发出“主角男的女的”的根本性质疑，严重破坏男性受众代入感；《狐作菲为》标为男频，实则用“化身狐妖靠美色勾引父子反目”的女频雌竞手段复仇，被直接拒稿。
*   **反派极度降智与脸谱化**
    *   *案例*：《出租狂飙》反派只会一口一个“死瘸子”和要求“跪下舔”；《被杀就满级》反派只会无脑叫嚣。纯粹为了提供嘲讽功能而存在的工具人反派，会大幅拉低剧本的高级感。

### 3. 和其他类型的关键差异

*   **主角必须具备绝对的“主观能动性”与“武力/智力压制”**：
    不同于女频可能存在的“被拯救”或“靠情感周旋”的情节，男频主角必须是破局的绝对主导者。例如《海王猎诡》将传统的“被诡异追杀”反转为“主角主动猎诡并收编为后宫”；而《狐作菲为》中靠出卖色相周旋复仇的方式在男频（甚至现代女频）是绝对的禁忌。
*   **情绪底色从“苦大仇深”向“轻松解构”转移**：
    传统的“灭门惨案/深山练功”（如《铁拳镇山河》）或“受辱复仇”（如《高武》）已不再吃香。男频受众现在更偏好用荒诞、搞笑、神豪等元素来解构严肃的冲突（如《南瓜成精》的植物拟人化）。
*   **对“数值/机制”的敏感度极高**：
    男频剧本通常需要一个清晰的“外挂机制”（如《高武》的余额兑换、《被杀就满级》的死亡升级），受众的爽感很大程度上来源于看着主角利用这些机制进行“降维打击”。

### 4. 评分修正建议（男频评审权重调整）

在评审“原创 / 男频”剧本时，建议对以下维度进行加/降权：

*   **大幅加权：`payoff_satisfaction`（爽点兑现）**
    *   *理由*：男频的生死线。只要打脸给得直接、极致、有画面感（如物理开颅、飞车比中指），即使其他方面有瑕疵，依然能获得高分（如《海王猎诡》）。
*   **加权：`premise_innovation`（设定新颖度）与 `pacing_conflict`（节奏与冲突）**
    *   *理由*：男频极度吃“脑洞”（如植物成精、余额战力），且要求极高的事件推进密度。前三集必须有明确的金手指展示和初次爽点释放。
*   **适度降权：`character_depth`（人物深度）**
    *   *理由*：虽然降智反派会被吐槽，但在男频爽文中，配角（尤其是女性角色和反派）的工具人属性是普遍存在的（如《海王猎诡》中的擦边女诡）。只要主角人设立得住、够杀伐果断，配角缺乏深度不应成为一票否决的理由。
*   **警惕项（一票否决）：性别视角混乱与绿帽/憋屈情节**
    *   *理由*：如《魅魔大叔》的男魂女身攻略男性，或主角长期被戴绿帽/被踩在脚下且迟迟不反杀，这类触碰男频受众核心雷区的情节，应直接给予极低分。

### 1. 这个类型特别看重什么（核心成功要素）

### 3. 可救 vs 不可救（“改”与“拒”的边界）

通过对比11部“改”和27部“拒”的剧本，责编的判别标准非常清晰：**“设定定生死，执行定改拒”。**

**🟢 什么是“差一点就能签”（可救）？**
