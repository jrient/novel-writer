# AI 短剧剧本创作手册（完整版）

> **用途**：给真人编剧 / Prompt 工程师阅读的 SOP。把「为什么」写清楚，便于人类调试与改写。
> **同步源**：基于 `handbook_v10.md`（85 部评审样本 + 57 部校准集）+ `writing_patterns.md`（16 部精品逆向）+ `archives/`（完整评审档案池）整合。
> **AI 注入版**：`ai_writer_prompt.compact.md`（精简硬规则，可直接塞 system prompt）。
>
> 修订原则：所有规则必须能在 v10 校准集里找到「签约 / 拒稿」对照证据，**没有证据就不写**。

## 0. 评分目标 —— 你在写哪一档？

校准集（57 部编辑确认）的实际分数分布：

| 状态 | 样本数 | 均分 | P25 | P75 |
|------|--------|------|-----|-----|
| **签** | 7 | 79.4 | 77.0 | 80.0 |
| 改 | 25 | 76.9 | 76.0 | 78.4 |
| 拒 | 25 | 72.9 | 70.0 | 74.7 |

- **签 / 改 边界 ≈ 78.2**（重叠区 77.0–78.4 取决于质性红/绿旗）
- **改 / 拒 边界 ≈ 74.9**（重叠区 74.7–76.0 取决于质性红/绿旗）

**目标**：写到 **78+ 才算精品**，76–78 是改稿生死线，<75 直接拒。
**最关键三维**（签 vs 拒差距最大）：**爽点兑现**（6.9 vs 5.0）、**人设立体度**（6.1 vs 4.2）、**对标差异化**（5.3 vs 3.4）。其他四维即便平均 5 分以上，差距不大；**先押这三维**。

## 一、七大评分维度详解

每个维度按「量化锚点 → 签约写法 → 死亡写法」三段呈现。

### 1. premise_innovation（题材与设定创新度）— 5.0/10

**量化锚点**

- ≤3 分 ≈ 100% 拒稿。直接照搬「闺蜜穿书 / 战神老祖下山 / 重生复仇 + 契约婚姻」会被一秒毙掉。
- 7-8 分能拿到极高编辑宽容度，但执行拉胯（如《高武》创新设定 + 苦大仇深错位）依然被拒。
- 微创新 = 在成熟赛道里找「反差感」。颠覆性世界观不是必需。

**签约写法**

| 案例 | 微创新点 |
|------|----------|
| 《天降魔丸》（80 分签） | 萌宝 + **反向带娃**（萌宝降维打击纨绔爹爹） |
| 《棺边低语》（81 分签） | 民俗悬疑 + **丧葬物件显遗言** |
| 《姐，咱家南瓜成精了》 | 修仙赛道 + **植物拟人成精**（颜文字蔬菜） |
| 《海王猎诡》 | 被诡追杀 → **主动猎诡收编后宫** |
| 《凤栖东厂》 | 古言 + **女主扮太监** |
| 《高考 750 分断绝关系》 | 男频「断绝关系流」+ **女频学霸设定** |

**死亡写法**

- 直接缝合两部已知爆款的设定。
- 把男频套路（战神 / 老祖 / 系统）性转给女频，导致「吵闹」「油」。
- 创新点挑战公序良俗（《狐作菲为》复仇逻辑崩坏 → 拒）。
- 高概念设定 + 错位情绪基调（荒诞设定却走苦大仇深 → 《高武》拒）。

### 2. opening_hook（开局与钩子）— 5.0/10

**量化锚点**

- 第 1 集决定生死，**前 1 分钟留不住观众，后期再精彩也无救**。
- 切入点过小（抢车位、收尸、喝咖啡重生）= 4 分以下。
- 第 1 集必须**同时**完成 5 件事（任缺一件 → 直接掉到 5 分以下）。

**签约写法 — 第 1 集 5 件套**

1. **5 秒内身份反差**：极度弱势 + 隐藏金手指（观众知道，剧中人不知道）。
2. **金手指实际效果展示**：不能只口头说，必须**演**出一次（南瓜的黄瓜祛斑 / 海王的金血秒杀 / 冷宫的随身空间制冰）。
3. **反派压脸**：主角必须被欺负 / 看不起 / 压迫一次（流氓栽赃 / 父亲扇耳光 / 婆婆逼吃诡饺）。
4. **钩子结尾**：最后一行不能是平淡场景，必须是悬念 / 转折 / 危机 / 新人物 / 反转 / 能力 / 情感 / 反派 / 预告 / 动作 10 种之一。
5. **性格特质展示**：通过具体行为而非台词，让观众一眼看到「主角是怎样的人」（《魔丸》一脚踹翻丹炉 / 《断绝关系》第 2 集签断绝协议）。

**死亡写法**

- 大空镜 + 缓慢建立场景 + 纯旁白开场。
- 男主第 5 集才出场（《诛仙台上捡个爹》被强烈要求提前）。
- 重生节点选在「平淡地喝咖啡 / 平淡地收尸」。
- 第 1 集只有金手指**口头**说明，没有实际效果（《满级刺绣》男主没看到女主绣工 → 拒）。
- 标题承诺的奇观（婆婆生三胎、72 套房）前 3 集没落地。

**钩子 10 类型轮换**（连续两集不能用同一种）

| 类型 | 典型句式 |
|------|----------|
| 悬念型 | "你，终于来了……" |
| 打断型 | 即将亲热 → 敲门声 |
| 危机型 | 菜地被浇柴油起火 |
| 反转型 | 傻子恢复记忆 |
| 新人物型 | 玛莎拉蒂 + 神秘女总裁出现 |
| 能力型 | 金龙刺青发光，植物反击 |
| 情感型 | 林秀云抱着流血的陈石头 |
| 反派型 | 神秘人攥住发光的菜 |
| 预告型 | "四大家族……满门抄斩" |
| 动作型 | 荆棘藤蔓如怒龙破土而出 |

### 3. character_depth（人设立体度）— 4.7/10 · **签 vs 拒最大差**

**量化锚点**

- 签均分 6.1，拒 4.2，差 1.9 分。**这是性价比最高的发力点**。
- 反派可以脸谱化，但**不能降智**。「在江城我就是王法」式的中二台词 → 直接扣分。
- 主角不能 5 集以上**只挨打不还手**（《离开闺蜜后》前 10 集纯挨打 → 拒）。

**签约写法 — 主角公式**

```
主角 = 表面标签 × 隐藏身份 × 标志性行为/台词
```

| 主角 | 表面 | 隐藏 | 标志行为 |
|------|------|------|----------|
| 南瓜男主 | 憨傻种地 | 仙农世家少主 | 跟蔬菜说话、护姐姐 |
| 毒妇女主 | 村中疯婆 | 顶级杀手穿越 | "天塌下来老娘扛" |
| 冷宫女主 | 冷宫废后 | 现代女 + 空间 | 卖小黄书"冤大头" |
| 八零福星 | 炮灰女二 | 商界女强人 | "我的人设从来都是自己给的" |

**反派必须有「可识别标志」**

| 反派 | 标志动作 | 标志台词 |
|------|----------|----------|
| 柳艳红 | 扭腰、拉低衣领 | "让姐姐摸摸" |
| 王彪 | 一脚踹门 | "老子村长的儿子" |
| 瑶妃 | 摔玉如意 | "本宫要她生不如死" |
| 皇甫绝 | 温润笑 + 算计 | "慕兮，过来坐" |

**每个女性角色必须有独立功能**（避免「漂亮 + 爱主角」工具人）

| 类型 | 角色定位 |
|------|----------|
| 靠山 / 资本 | 南瓜的沈清辞、海王的楚曼 |
| 武力担当 | 海王的姜红衣 |
| 智囊 | 冷宫的银霜 |
| 情感冲突核心 | 嫡女的堂妹 |
| 搞笑担当 | 下山祸害的二师姐 |

**死亡写法**

- 反派连续 3 集只会扇耳光 + 喊口号（《妈妈，这次换我爱你》医院走廊三集辱骂 → 拒）。
- 反派第 7 集就被抓，剩下 53 集没有同级威胁（《穿书炮灰》→ 拒）。
- 主角被系统 / 剧本逼着违心做事（《黑月光》女主被逼当恶毒女配 → 阅读体验憋屈）。
- 「精神病人的幻想」式终极反转（《和男友父亲同棺》→ 烂尾、必拒）。

### 4. pacing_conflict（节奏与冲突密度）— 5.2/10

**量化锚点**

- 每集冲突 ≥ 2 次，打脸 ≥ 1 次。
- 单一场景拉扯**不能超过 2 集**（《被抢车位后》8 集都在一个车位 → 拒）。
- 任何超过 5 行的纯描述段落都是节奏拖沓。
- 30 秒内必须有冲突，否则观众划走。

**单集三段式**

```
开场 (5-15s)：1-2 句台词建立当下情境，最好直接切入冲突
冲突升级 (15-90s)：≥1 个对抗 / 打脸事件
钩子结尾 (90s~end)：最后一行必须是悬念 / 转折 / 动作高潮
```

**前 10 集（一卡）弧线**

```
第 1 集：世界观 + 人设 + 金手指亮相（必须爽）
第 2 集：小冲突验证能力（爽感巩固）
第 3 集：更大冲突 / 新反派登场（拉高期待）
第 4-5 集：连续冲突链，每集一个"打脸 / 反杀"
第 6-7 集：中期 BOSS 登场，能力被压制 / 考验
第 8-9 集：身份暴露 / 马甲松动 / 感情升温
第 10 集：一卡高潮，大反转 / 大悬念（逼付费）
```

**爽-紧-悬循环**

```
爽（主角展示能力 / 反击 / 打脸）
  → 紧（反派反扑 / 新危机出现）
    → 悬（结尾留钩子，观众想看下一集）
```

**死亡写法**

- 单一场景反派单方面输出 3 集以上。
- 全集只有日常 / 背景交代，没有冲突事件。
- 高潮场景突然「主角气晕倒下」「反派代为受过」（《高考 750 分》女主气晕 → 阻断爽感）。

### 5. writing_dialogue（文笔与台词）— 4.8/10

**量化锚点**

- 单集 20-40 句对话；OS 占比 20-30%。
- **绝大多数台词 < 15 字**。
- 长段独白（连续 3 句以上同角色）= 拖沓。

**签约写法**

- **OS 补动机**，不补动作（解释「为什么这么做」，而不是描述「正在做什么」）。
- **视觉化描写**：把「她很伤心」改成「她眼眶红了，没说话」。
- 短句 + 情绪括号叠加：`角色名（隐忍 → 崩溃）：台词`。
- 每人连续对白 ≤ 3 句，中间穿插动作或对方反应。

**正例**

```
✅ 林秀云 OS：这些年为了给石头看脑袋，家里入不敷出……
✅ 叶慕兮 OS：前世他就是借着这场踏青宴，毁了我的名声……这一世，我定要让他自食恶果。
✅ 陆云萝（冷笑）：我可以给你 20 个大逼兜，想试试吗？
```

**死亡写法（红线）**

- **古装语境塞现代网络词**（一票否决项）：
    - "大逼兜 / 视觉污染 / 现场直播 / 金主爸爸 / 别墅 / 老铁 / 我靠 / 天啦噜 / 祖国大大 / 一分钟啊大哥 / 老娘"
- 「她感到一阵苦涩涌上心头」式的全知心理。
- 排比 / 三段论：连续三个结构相似的句子或对白。
- 万能动词 "感到 / 觉得 / 变得 / 充满"。
- 每次说话前先念对方名字（《捉奸现场》→ 不自然）。

### 6. payoff_satisfaction（爽点兑现）— 5.5/10 · **签 vs 拒最大差**

**量化锚点**

- 签均分 6.9，拒 5.0，差 1.9 分。
- **爽点必须当场兑现**，禁止「下一集再报复」。
- 每 10 集需出现以下爽点各 1 次：能力碾压、身份打脸、财富展示、情感推进、打脸造谣者。

**打脸的 4 种标准写法**

| 写法 | 例子 |
|------|------|
| 直接物理打脸 | 一巴掌扇出血、20 个大逼兜、银霜扇巴掌 |
| 能力碾压打脸 | 金龙刺青发光秒杀、九曜连珠穿心 |
| 身份反转打脸 | 恶魔印记 + 总裁身份、面具下真容 |
| 证据 / 信息差打脸 | 账本打脸贪墨、预知基金暴雷、银针验毒 |

**欲扬先抑三步法**（每 10 集 ≥ 3 次）

```
Step 1: 主角被极度羞辱（被骂傻子 / 被扇耳光 / 被看不起）
Step 2: 旁观者嘲笑 / 嘲讽 / 补刀
Step 3: 主角反击，围观者震惊 / 打脸
```

**信息差打脸升级**：让敌人自己吃自己下的药（《下山祸害》：齐天宇下药 → 狗调包酒杯 → 齐天宇喝下自己的药）。

**死亡写法**

- 「下一集再报复」「忍辱负重等时机」式拖延。
- 高潮处妥协 / 转移惩罚（嬷嬷代为受过、突然气晕）。
- 反派过早溃败导致后续 30+ 集没主线敌人。
- 主角被动等待救援。

### 7. benchmark_differentiation（对标与差异化）— 4.6/10 · **签 vs 拒最大差**

**量化锚点**

- 签均分 5.3，拒 3.4，差 **1.9** 分。
- 「集齐了市面热门元素（马甲 + 双强 + 萌宝 + 带球跑）」**已经不再是优势**。
- 必须能用一句话说清「我跟 XX 的核心区别是什么」。

**签约写法**

- 在剧本/大纲首页明确标注**对标作品 + 差异点**。
- 差异点必须是**结构性**的（如反向带娃 / 主动猎诡 / 反向婆媳），不是装饰性的（颜值更高 / 服化更精）。

**死亡写法**

- 直接对标抖音剧情号但反转力度更弱（《穷鬼的助学金》→ 被指出抖音有同剧）。
- 「平平无奇大女主」（《嫡女贵凰》虽签但被点名 → 危险信号）。
- 既视感强（《重回八零护女反杀》"在其他渠道看过，很诡异" → 拒）。

## 二、类型专项手册

> 写之前必须先确定类型。**类型决定加权与红线**。
> 类型分类与 v10 校准集一致。

### A. 萌宝（原创 / 萌宝）

**核心加分项**

- 萌宝**反差感**：好赌、海量、反向带娃。常规乖巧奶团子已无市场。
- 开局即建立**核心家庭关系**（带球跑 / 天降 / 认亲），不能只做技能展示。
- **女主必须有独立高光**，不能因为萌宝抢戏而隐形。

**必拒红线**

- 萌宝技能堆砌但无微创新（下山 + 算命 + 找首富爹）。
- 男主 5 集后才出场。
- 反派降智 + 物理攻击水时长（连续 3 集医院走廊扇耳光）。

**评审权重**：premise_innovation 大幅加权，opening_hook 加权，benchmark_differentiation 中"常规元素堆砌"降权。

---

### B. 改编 / 女频

**核心加分项**

- **爽点兑现极速 + 不内耗**：当场反击、当场获利。
- **CP 化学反应**：阳光开朗金毛 × 阴湿大猫式反差，男女主前 10 集必须有强宿命互动。
- **女主现代独立感**：搞钱 / 高智 / 武力，拒绝古早小白花。
- **跨频微创新**：把男频「断绝关系流」拿来配女频学霸 → 惊艳。

**必拒红线**

- 古装语境滥用现代网络词（一票否决）。
- 反派强行降智为送人头。
- 高潮处妥协导致憋屈。
- 过度依赖金手指（如随身空间）解决一切，主角无主观能动性。

**评审权重**：payoff_satisfaction、character_depth 加权；premise_innovation 降权（赛道成熟）。

---

### C. 改编 / 世情

**核心加分项**

- **冲击力社会议题或奇观设定**：高龄婆婆生三胎、天降 72 套房、巨款被骗。
- **快节奏闭环**：受气 → 爆发 → 反转 必须在 2-3 集内完成。
- **逻辑严密**（叠加重生等设定时不能漏洞）。

**必拒红线**

- 触碰伦理底线 / 价值观扭曲（放弃赡养、报复反人类）。
- 全员恶人式极端脸谱化。
- 核心卖点（婆婆怀孕等）3 集内不落地。
- 与抖音剧情号既视感重合且反转更弱。

**评审权重**：pacing_conflict、payoff_satisfaction、benchmark_differentiation 加权；writing_dialogue 降权（直白即可）。
**红线**：价值观一票否决。

---

### D. 改编 / 男频

**核心加分项**

- **硬核写实技能**胜于夸张悬浮战斗（贝爷式求生 > 鲨鱼搏斗）。
- **极致阶级反转**（老板下线，孤岛重建秩序）。
- **打破常规的差异化人设**（亦正亦邪 / 朋克武侠）。

**必拒红线**

- 常识错误：联邦 / 军衔倒挂 / 历史穿帮。
- 开局过早揭底，丧失悬念。
- 中后期单一孤岛设定后继乏力，无新危机引入。
- 触碰审查红线（军区、政府题材未谨慎处理）。

**评审权重**：premise_innovation、benchmark_differentiation、pacing_conflict 加权；payoff_satisfaction 降权（华而不实视觉奇观要扣）。

---

### E. 原创 / 男频

**核心加分项**

- **极致直白爽点**：物理开颅、一击秒杀、数值碾压（战力 = 银行卡余额）。
- **微创新反差**：植物成精、主动猎诡、出租车飞跃。
- **极快冲突推进**：第 10 集前完成多次大战与升级。
- **强烈视觉刺激**：腹肌、动作戏、奇观。

**必拒红线**

- 古早开局：「测天赋废柴 + 众人嘲笑 + 拜金女友分手」。
- 设定与基调错位：荒诞金手指 + 苦大仇深复仇。
- 反派降智 + 古早作恶（剪刹车线、口吐芬芳）。
- 受众错乱：标男频写女频内核（《狐作菲为》《我和魅魔大叔》）。

**评审权重**：payoff_satisfaction、premise_innovation 加权；character_depth 降权（脸谱化可接受）。

---

### F. 原创 / 女频

**核心加分项**

- **性张力与情感拉扯**：成年人对话感、反差魅力、宿命感。
- **爽点前置**：女主必须在前 3 集破局，不能纯挨打。
- **反差 / 猎奇微创新**：女主扮太监、婆媳同孕、反向婆媳。
- **金手指视觉化 + 互动化**：男主必须看到女主用技能。

**必拒红线**

- 男频套路硬核元素（战神性转、闪电战 / 蛙跳战术大段描写）。
- 系统逼迫女主作恶 + OS 立牌坊。
- 海王 / 恶女人设无道德挽回 / 高光。
- 网文古早语：天啦噜、祖国大大、说话先念对方名字。

**评审权重**：CP 化学反应、开局情绪钩子、人设微创新加权；硬核世界观、被动型主角大幅降权。

---

### G. 原创 / 世情

**核心加分项**

- **前 3 集极致情绪起搏**：前世惨死 → 今生关键转折点；重生打脸闺蜜要钱。
- **打脸密集小爽点**：每集至少 1 次反派挑衅 → 主角反击 → 反派受惩闭环。
- **经典框架 + 差异化标签**：重生 + 赛车 / 重生 + 民俗。
- **反派要有段位**：必须能撑到中后期。

**必拒红线**

- 前期憋屈太久（8 集才爆发）。
- 后期反派报应过快导致剧情乏力。
- 突兀肢体冲突（无铺垫的扇耳光）。
- 同质化既视感（重返 82 / 重生九零等模板）。

**评审权重**：opening_hook、payoff_satisfaction、pacing_conflict 加权；premise_innovation、writing_dialogue 适度降权。

## 三、反 AI 味 10 条（与代码 `style_guard.ANTI_SLOP_RULES` 同步）

绝对不要出现以下内容：

1. **过度抽象形容词**："眼神中透露出坚毅的目光" / "无法言喻的复杂情绪"
2. **套路化比喻 / 暗喻**："仿佛一把利刃刺穿了心" / "月光如水"
3. **书面语对白**：角色说话要像真人日常对话，不像论文 / 演讲
4. **环境描写先行开场**：第一镜禁止大空镜 / 缓慢建立场景 / 纯旁白开场
5. **情绪解释代替情绪表现**：写"她眼眶红了，没说话"，不写"她感到很伤心"
6. **总结性陈词与升华**：不写"这一刻他明白了勇气的真谛"，留白处理
7. **万能动词** "感到 / 觉得 / 变得 / 充满"：用具体阻力动词替代（"指关节捏得咯吱响"）
8. **排比 / 三段论**：禁止连续三个结构相似的句子或对白
9. **全知视角心理描写**："他心中泛起一丝苦涩" → 用镜头可见的微表情 / 动作替代
10. **分屏镜头**：禁止出现"画面左右分屏 / 一分为二"，所有镜头必须是完整单屏

## 四、地雷清单 TOP 10（一票否决组合）

来自 v10 训练集 50 部拒/改剧本的高频拒稿原因。任一出现 → 直接掉档。

1. **古装 / 奇幻背景滥用现代网络烂梗**（"大逼兜"、"视觉污染"、"现场直播"）
2. **核心冲突极度拖沓，单一场景水字数**（一个车位 8 集）
3. **反派全员降智，沦为只会叫嚣的脸谱化工具人**
4. **开局缺乏情绪钩子，进入主线太慢**（喝咖啡重生 / 平淡收尸）
5. **题材严重同质化，缺乏任何微创新**（重生复仇 + 契约婚姻 / 萌宝下山找爹）
6. **主角缺乏主观能动性**（被系统 / 剧情推着走）
7. **核心设定逻辑硬伤或常识匮乏**（汉代找《中国古代史》/ 漠北沼泽 / 残疾不影响台球）
8. **受众定位错乱**（男 / 女频爽点生搬硬套，性转吵闹）
9. **爽点兑现严重滞后**（前 10 集纯隐忍 / 阶段性释放缺失）
10. **滥用闪回与内心独白，叙事结构稀碎**

### 致命组合（必拒套餐）

- **老旧套路 + 现代烂梗** = 廉价缝合怪
- **猎奇噱头 + 烂尾底牌** = 欺诈观众（精神病幻想式反转）
- **系统逼迫作恶 + 主角内心吐槽** = 精神分裂
- **单一场景 + 反派单方面输出** = 终极水字数

## 五、单集字数 / 时长 / 输出格式

### 字数

| 指标 | 男频 | 女频 | 通用 |
|------|------|------|------|
| 每集行数 | 20-40 | 15-30 | 15-40 |
| 每集字数 | 600-1000 | 400-800 | 400-1000 |
| 单场景行数 | 8-20 | 5-15 | 5-20 |
| 前 10 集总字数 | 8k-10k | 6k-8k | 6k-10k |

### 输出格式（动态漫 / 真人剧）

```
1-1 场景名 时间 内/外
人物：角色 A、角色 B、角色 C
△ 动作 / 场景描写（不超过 5 行）
角色 A（情绪）：台词
角色 B（OS）：内心独白
【闪回】（可选）
【字幕】（可选）
```

- `△` 动作 / 场景描写
- `（OS）` 内心独白
- `VO` 画外音 / 旁白
- 最后一行必须留悬念

## 六、第 10 集（一卡卡点）必须满足

1. **大高潮**：前 9 集积累的矛盾在此爆发
2. **新悬念**：揭示更大威胁 / 更强敌人
3. **身份变化**：主角获得新身份 / 新能力 / 新关系
4. **情感高潮**：男女主感情推进或重大转折

## 七、自检清单（提交前过一遍）

**开局（第 1 集）**
- [ ] 5 秒内出现身份反差
- [ ] 金手指有实际效果展示，非口头
- [ ] 主角被反派压脸 ≥ 1 次
- [ ] 钩子结尾不重复前一集类型
- [ ] 通过具体行为展示主角性格

**节奏（每集）**
- [ ] 冲突 ≥ 2 次，打脸 ≥ 1 次
- [ ] 30 秒内必有冲突
- [ ] 没有 5 行以上的纯描述
- [ ] 钩子结尾非平淡场景

**对白**
- [ ] 绝大多数台词 < 15 字
- [ ] OS 占 20-30%，只补动机不补动作
- [ ] 没有古装语境的现代网络词
- [ ] 没有连续 3 句以上的独白

**爽点**
- [ ] 当场兑现，没有「下一集再报复」
- [ ] 每 10 集 4 种打脸方式都用过
- [ ] 欲扬先抑三步法 ≥ 3 次

**人设**
- [ ] 反派有可识别的标志动作或台词
- [ ] 反派不降智（行为有逻辑）
- [ ] 每个女性角色有独立功能
- [ ] 主角不是 5 集以上只挨打不还手

**类型 / 红线**
- [ ] 已确定类型，按对应类型专项加权
- [ ] 没有触碰类型对应红线（如世情伦理、男频常识）
- [ ] 没有反 AI 味 10 条
- [ ] 没有地雷 TOP 10 组合

**对标差异化**
- [ ] 一句话说清「跟对标作品的核心区别」
- [ ] 区别是结构性的（反向 / 跨频 / 设定颠覆），非装饰性
