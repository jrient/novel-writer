## 第一部分：通用规律

基于提供的33部剧本评审档案摘要，我为您提取了短剧剧本在7大核心维度的规律与数据洞察。以下是首席分析师视角的结构化报告：

### 1. premise_innovation（题材与设定创新度）

*   **核心规律**：
    *   创新必须建立在市场认知基础之上，认知门槛过高或价值观模糊的设定会被直接排斥。
    *   “高概念”设定是获取责编修改意愿（“改”状态）的最强护身符，哪怕执行拉胯也能获得机会。
    *   仅靠叠加职业标签（如刺绣、种田）作为噱头，而内核依然是老套路的“伪创新”，无法逃避被拒的命运。
*   **正例（签）**：
    *   《海王猎诡：我把替身船玩成了后宫》（签）：设定极具想象力与“邪典”属性，直接拉满情绪期待，获得8分高分。
*   **反例（拒）**：
    *   《满级刺绣一卡》（拒）：刺绣元素仅为噱头，本质依然是套路化的普通言情，被识破后直接拒稿。
    *   《高武：我的战力是银行卡余额》（拒）：虽然设定新颖（7分），但强行套用古早老套的严肃复仇流写法，设定与执行严重割裂。
*   **量化锚点**：
    *   维度分 ≤ 3 的剧本（如《萌宝8岁》《秦总对着我的遗照说爱我》）签约率为 0%。
    *   维度分 ≥ 7 的剧本，即便其他维度存在硬伤，获得“改”（修改机会）的概率高达 85% 以上。
*   **可执行建议**：
    *   **Do**：将创新点深度融入主线剧情与核心爽点中，让设定直接为情绪服务（如喜剧、颠感）。
    *   **Don't**：不要做“换皮不换骨”的微创新，也不要挑战受众的伦理认知底线。

---

### 2. opening_hook（开局与钩子）

*   **核心规律**：
    *   开局必须迅速建立情绪共鸣或高压冲突，慢热是短剧的绝对死穴。
    *   男女主之间的“宿命感”或核心矛盾必须在开局迅速确立。
    *   复杂的背景交代或平淡的重生节点会直接导致读者（责编）流失。
*   **正例（签）**：
    *   《海王猎诡：我把替身船玩成了后宫》（签）：开局钩子高达8分，节奏快、情绪调动直接，迅速抓住眼球。
*   **反例（拒）**：
    *   《二嫁权宦》（拒）：开局太平淡，缺乏情绪调动，男女主早期缺乏宿命感与互动。
    *   《ai仿真人 撕碎闺蜜后，我成了靳总心尖宠》（拒）：开局进入情绪太慢，重生节点平淡，无法留客。
*   **量化锚点**：
    *   开局与钩子分 ≤ 4 的剧本，除非是自带流量的成熟IP改编，否则原创剧本淘汰率接近 100%。
*   **可执行建议**：
    *   **Do**：把全剧最具爆发力的冲突或最极致的情绪点前置到第一集前30秒。
    *   **Don't**：不要在开局安排大量闪回（如《凤栖东厂》的负面反馈）或平铺直叙的背景介绍。

### 3. character_depth（人设立体度）

*   **核心规律**：
    *   主角必须具备极强的主观能动性，绝不能被系统或剧情推着走。
    *   反派不能是纯粹的“工具人”或脸谱化降智角色，这会拉低整体剧情的质感。
    *   主角行为逻辑必须符合其身份设定，不能为了制造冲突而强行降智。
*   **正例（签）**：
    *   《八零福星俏娇妻》（签）：人设扎实（6分），获得了责编“挑不出毛病、非常成熟”的高度评价。
*   **反例（拒）**：
    *   《被杀就满级，我乃皇家第一侍卫》（拒）：男主缺心眼，女主没脑子，反派脸谱化，角色智商全员不在线。

## 第二部分：类型专项

### 原创 / 萌宝

基于提供的三部“原创 / 萌宝”剧本评审档案，以下是该类型剧本的独特要求、常见翻车点及评审标准修正建议：

### 1. 这个类型特别看重什么？

*   **女主的主视角地位与独立高光**
    萌宝虽然是核心卖点和“最强助攻”，但绝不能喧宾夺主让女主沦为工具人。观众期待看到女主自身的强势与魅力。
    *   *案例*：《诛仙台上捡个爹》中，评审明确指出需要“主视角改为女主”，并强调“希望之后女主高光多一些”，不能仅靠萌宝推进剧情。
*   **核心人物关系（男女主）的快速建立**
    短剧节奏要求极高，萌宝文的看点在于“一家三口”的互动与拉扯，男主出场绝对不能晚。
    *   *案例*：《诛仙台上捡个爹》因“男主出场有点晚，第五集才出来”被多位评审诟病，严重削弱了前期的期待感。
*   **极致但符合逻辑的“虐点”与情绪共鸣**
    下沉市场非常吃“萌宝护父/母”的虐心情节，但这种“惨”必须建立在合理的基础之上，不能为了虐而强行制造苦难。
    *   *案例*：《妈妈，这次换我爱你》中“萌宝护父的虐心点”成功抓住了受众情绪，但因设定不合逻辑（腿残疾的台球高手却只能靠女儿捡废品为生）而遭到严重质疑。
*   **热门元素的叠加与细腻的感情戏**
    在常规套路之上，精准堆叠下沉市场喜爱的元素，并辅以细腻的感情描写，能有效提升剧本质感。
    *   *案例*：《诛仙台上捡个爹》集齐了“马甲、奶团子、双强、带球跑、双向奔赴”等元素，配合“细腻的感情戏”，获得了部分评审“会好看”的高预期。

### 2. 这个类型的常见翻车点

*   **高度同质化，缺乏核心差异设定**
    萌宝赛道极度内卷，如果只是套用经典模板，极易被市场淘汰。
    *   *案例*：《萌宝8岁，下山被大佬爹地宠上天》因“几乎也是同质化的内容，创新点不多”被拒；《诛仙台上捡个爹》也被指出“重复度有点太高了，萌宝需区别于市面”。
*   **开局钩子弱，错把“技能展示”当“剧情冲突”**
    第一集必须有强冲突卡点，不能仅仅为了展现萌宝的可爱或特殊能力而平铺直叙。
    *   *案例*：《萌宝8岁》第一集“只表现萌宝会算命，卡点差点意思”，导致开局缺乏吸引力。
*   **反派强行降智，冲突拖沓重复**
    为了衬托萌宝的聪明或主角的惨，把反派写得极度脸谱化，且用单一的手段（如扇耳光）水时长。
    *   *案例*：《妈妈，这次换我爱你》中反派台词极度中二（“在江城我就是王法”），且连续三集（8-10集）在医院走廊上演单方面辱骂扇耳光的戏码，被批极度拖沓。
*   **职业元素与常识脱节**
    为了微创新给主角叠加职业属性（如体育竞技），但作者缺乏相关知识储备，导致剧情露怯。
    *   *案例*：《妈妈，这次换我爱你》加入了台球事业元素，但被评审指出“台球知识好浅薄”，且职业属性与残疾设定产生严重逻辑冲突。

### 3. 和其他类型的关键差异

*   **“双核驱动”的爽点机制**：与纯战神或纯女频虐恋不同，萌宝题材的爽点是双核的——既需要“萌宝花式助攻/打脸”的趣味性爽点，又必须回归到“男女主双强/高光反击”的传统言情爽点，两者缺一不可。
*   **对“微创新”的容错率极低**：由于“带球跑、找首富爹地”等大框架已被写透，该类型比其他类型更依赖**人设微创新**（如萌宝的特殊性格/技能，或父母的特殊职业）。如果没有差异化，即使文笔合格也会因为“重复度太高”被直接毙掉。

### 4. 评分修正建议（针对“原创 / 萌宝”类型）

在评审此类剧本时，建议对以下维度进行权重调整：

*   **加权维度：**
    *   **【设定创新/差异化】（大幅加权）**：鉴于赛道饱和，只要萌宝人设或父母互动模式有亮眼的微创新，应给予高分鼓励；若纯属同质化套路，应直接打低分。
    *   **【开局钩子】（加权）**：重点考察男主出场速度（前3集内必须建立核心羁绊）以及第一集是否有实质性冲突（而非单纯展示萌宝技能）。
    *   **【人物塑造】（加权）**：重点考察**女主**是否拥有独立高光和主视角地位，以及父母颜值/人设的基础吸引力。
*   **降权/严打维度：**
    *   **【节奏与冲突】（严格审视）**：严打利用反派无脑叫嚣、连续多集扇耳光/辱骂来水时长的行为。爽点兑现（Payoff）不能过度滞后。
    *   **【逻辑与常识】（严格审视）**：严打为了“虐”而强行制造的逻辑硬伤（如《妈妈》中违背常理的谋生方式），以及专业领域知识的匮乏。

### 改编 / 女频

基于提供的9部“改编 / 女频”剧本评审档案，以下是该类型剧本的通用规律与独特要求总结：

### 1. 这个类型特别看重什么？

*   **爽点兑现的速度与暴力直接的反馈**
    女频爽文改编极度看重主角面对挑衅时的反击效率和力度，不能有无效拉扯。
    *   *案例*：《八零福星俏娇妻》中女主面对造黄谣的反派，直接采取“物理+言语双重打击”（踹窝心脚+挥舞棒槌抽打），爽点充足，获得签约；《嫡女贵凰》中女主直接将茶水泼在反派身上、当场捉奸，爽点兑现极快。
*   **极致且稳定的大女主/强人设气场**
    主角必须时刻保持清醒、独立或霸气，任何软弱、迟钝或与人设不符的咋咋呼呼都会严重破坏观众代入感。
    *   *案例*：《八零福星俏娇妻》中女主台词“对垃圾回收不感兴趣”有效立住了女强人气场；但其开局遇到人贩子时大喊大叫的反应，被责编批评与“冷静商界女强人”人设割裂。
*   **男女主早期的宿命感与高频互动**
    即使是主打剧情/宅斗的女频剧，感情线也是核心刚需。男主必须尽早出场，且具备极强的存在感或极致人设（如霸道、权臣）。
    *   *案例*：《二嫁权宦》因“男女主前期对手戏太少”、“男主不够霸气”、“缺少宿命感”被拒稿；已签约的《一世容安》也被责编明确指出“男主前10集存在度太低，需要往里塞塞”。
*   **利用“信息差/金手指”制造直观的阶级反差**
    女频穿越/重生题材非常看重主角利用现代知识或前世记忆对古人/反派进行“降维打击”，并形成强烈的视觉或境遇对比。
    *   *案例*：多部《我在冷宫忙种田》的改编中，女主利用空间制冰，在冷宫里吹冷气吃冰，与外面热得扇扇子的皇帝形成平行剪辑对比，这是该剧本为数不多被认可的爽点机制。

### 2. 这个类型的常见翻车点

*   **台词过度现代网络化，严重破坏古言语境**
    在古言穿越中，为了制造笑点而滥用现代网络词汇，会导致剧本充满“小白感”和悬浮感，极易引发观众出戏。
    *   *案例*：多部《我在冷宫忙种田》废稿中出现了“金主爸爸”、“我靠”、“油腻男”、“现场直播”、“视觉污染”、“一分钟”等台词，被责编集体诟病。
*   **反派强行降智，沦为纯粹的“送分工具人”**
    为了衬托女主的聪明而把反派写得毫无逻辑、动机单一，导致博弈缺乏张力，爽感大打折扣。
    *   *案例*：《帝少的重生毒妻》中的简淑念被直接设定为“智商不高，嚣张跋扈”；《我在冷宫忙种田》中的反派纯粹为了送人头而存在，被批“脸谱化严重”。
*   **开局铺垫过长，缺乏即时的情绪钩子**
    女频受众缺乏耐心，如果第一集还在交代前史或进行表面寒暄，没有生死危机或强烈的情绪刺激，很容易流失观众。
    *   *案例*：《八零福星俏娇妻》第一集现代戏占了一半，女主确认穿书的反射弧过长被批拖沓；《二嫁权宦》开局只有一个收尸情节，被批“太平、情绪调动感不够”。
*   **副本推进过快，出场人物过载**
    虽然要求节奏快，但如果在极短的集数内塞入过多反派和事件，会导致观众认知疲劳。
    *   *案例*：《嫡女贵凰》前10集塞入了三个副本（解决叶婉芙、救弟弟查贪墨、桃花别院反杀），出场人物过多，被责编指出“导致人看得有点累”。

### 3. 和其他类型的关键差异

*   **对“套路”的包容度极高，执行力大于原创性**
    与其他类型苦求“设定创新”不同，女频改编（如重生宅斗、穿书年代文）的受众基本盘非常稳固，**不怕套路老，就怕爽点不到位**。例如签约的《嫡女贵凰》和《一世容安》，其 `premise_innovation`（设定创新）得分仅为 4 分和 6 分，被评为“平平无奇大女主”、“非常常规”，但因为打脸节奏极好（爽点8分），依然顺利签约。
*   **“事业（复仇）”与“情感（CP）”必须双轴驱动**
    纯男频可能只需一路升级打怪，但女频改编中，女主的“搞事业/复仇反杀”必须与男主的“追妻/双强/宿命纠缠”交织进行。单线叙事（如男主长期掉线）会被视为重大结构缺陷。

### 4. 评分修正建议（针对“改编 / 女频”类型）

在评审此类型剧本时，建议对以下维度进行权重调整：

*   **大幅加权：`payoff_satisfaction`（爽点兑现）**
    *   *理由*：这是该类型的生死线。档案显示，只要该维度达到 8 分（如《八零福星俏娇妻》、《一世容安》、《嫡女贵凰》），即使设定老套、人物稍多，也全部获得了“签”的状态。
*   **适度加权：`pacing_conflict`（节奏与冲突）**
    *   *理由*：单集内必须有明确的打脸或危机推进，不能有无效的日常水戏。
*   **大幅降权：`premise_innovation`（设定创新）与 `benchmark_differentiation`（竞品差异化）**
    *   *理由*：该赛道高度成熟（如“重生+复仇+宅斗”、“穿书+八零+炮灰逆袭”），同质化是常态。不应因为“题材常见”、“套路古早”就直接毙掉剧本，只要微创新（如人设更极致、台词更利落）到位，依然具备极高的商业价值。
*   **严格审查（一票否决项）：`writing_dialogue`（台词文笔）中的“语境破坏”**
    *   *理由*：古言题材中一旦出现大量现代网络烂梗（参考《我在冷宫忙种田》的多次拒稿），应直接判定为不合格，这代表编剧缺乏基本的题材把控力。

### 原创 / 男频

基于提供的8部剧本评审档案，针对“原创 / 男频”这一类型的通用规律、独特要求及评审标准，总结如下：

### 1. 这个类型特别看重什么？

男频短剧极度追求**即时刺激、视觉奇观与情绪的绝对释放**，具体表现在以下几个核心维度：

*   **极致且具画面感的爽点兑现（视觉冲击力）**
    *   男频的“爽”不能仅停留在口头，必须有强烈的动作设计或视觉奇观来承载。
    *   *案例*：《ai仿真人 出租狂飙》中，核心爽点通过“出租车飞出赛道在空中超越顶级赛车并比中指”来兑现，画面感极强；《姐，咱家南瓜成精了！》中，通过“吃苦瓜放屁”、“黑芝麻瞬间生发”等极具恶搞趣味的视觉效果立竿见影地打脸。
*   **极快的节奏与高密度的冲突（拒绝憋屈）**
    *   男频受众对“压抑”的容忍度极低，要求主角迅速反杀、持续升级或转换地图。

### 2. 这个类型的常见翻车点

*   **古早套路开局与反派极度降智（刻板脸谱化）**

*   **基调错位，浪费核心设定**
    *   想出了一个极具喜剧或荒诞色彩的好设定，却非要套用“苦大仇深”的传统复仇模板。
    *   *案例*：《高武：我的战力是银行卡余额》中“1元=1战力”本是一个极具网感、适合走轻喜剧/神豪装逼风的设定，作者却用它来写传统的受辱复仇流，导致严重违和。
*   **受众定位模糊与性别认知障碍**

### 3. 和其他类型的关键差异

*   **对“暴力/擦边/粗口”等下沉元素的包容度极高，但需警惕红线：** 相比女频的情感拉扯，男频更追求直接的感官刺激。例如《海王猎诡》中大量使用“老子”、物理开颅、制服女诡等元素，虽被认为“有劲”，但也伴随着极高的平台审核风险（物化女性、血腥暴力）。

*   **世界观的“壳子”必须能承载设定的“颠”：** 男频经常引入系统、异能等元素，但必须与背景融合。例如《被杀就满级》在古装宫廷权谋中滥用“牛马、老登、绩效”等现代词汇，引发严重出戏；而《南瓜成精》虽然颠，但放在乡土奇幻的壳子里就显得合理且上头。

### 4. 评分修正建议（评审加/降权维度）

在评审“原创 / 男频”剧本时，建议对以下维度进行权重调整：

*   **【大幅加权】爽点兑现（Payoff Satisfaction）：**
    *   只要爽点给得直接、极致、画面感强（如飞车比中指、物理开颅、吃苦瓜放屁），即使文笔稍弱也应给予高分。这是男频短剧的生命线。
*   **【大幅加权】节奏与冲突密度（Pacing & Conflict）：**

*   **【适度加权】设定创新度（Premise Innovation）：**
    *   对带有“邪典”、“轻喜”、“反套路”标签的微创新（如战力=余额、植物成精）给予鼓励分，男频市场极度渴求新鲜的“钩子”。
*   **【适度降权】反派人物深度（Character Depth - Villains）：**
    *   男频短剧中，反派的“工具人”属性是普遍存在的。只要反派的降智行为没有严重破坏世界观逻辑（如《被杀就满级》中女帝无脑倒贴杂役），单纯的“嚣张跋扈”不应作为致命扣分项，应将重点放在主角的反杀手段是否精彩上。

### 原创 / 女频

### 1. 这个类型特别看重什么？

*   **极致的设定与反差感（Premise Innovation）**
    女频受众极度渴望新鲜感，一个打破常规、带有禁忌感或奇幻色彩的设定能迅速抓住眼球，甚至掩盖部分文笔瑕疵。

*   **男女主之间高质量的情感拉扯与性张力（Chemistry & Tension）**
    女频剧本的核心看点在于成年人之间的恋爱博弈，人设必须有魅力，互动必须有“涩气”或宿命感。

### 2. 这个类型的常见翻车点

*   **文笔小白，台词与人设低幼化**
    现代情感剧如果缺乏成年人语境，男主表现得像个毛头小子，会引发强烈的出戏感和反感。

*   **开局情绪太平，缺乏有效钩子**
