## 二、集数结构模板

### 单集内部结构（固定"三段式"）

```
开场(5-15s)  →  冲突升级(15-90s)  →  钩子结尾(90s~结束)
```

**铁律：每一集的最后一行一定是悬念/转折/动作高潮，绝不以平淡场景收尾。**

### 单集字数与行数

| 指标 | 范围 | 中位数 |
|------|------|--------|
| 每集行数 | 15-40 | 25 |
| 每集字数 | 400-1000 | 650 |
| 单场景行数 | 5-20 | 12 |
| 每集对话数 | 20-40句 | 28 |
| OS占比 | 20-30% | 25% |

### 前10集（一卡）完整弧线

```
第1集：世界观+人设+金手指亮相（必须爽）
第2集：小冲突验证能力（爽感巩固）
第3集：更大的冲突/反派登场（拉高期待）
第4-5集：连续冲突链，每集一个"打脸/反杀"小高潮
第6-7集：中期BOSS登场，能力被压制/考验
第8-9集：身份暴露/马甲松动/感情升温
第10集：一卡高潮，大反转/大悬念，逼观众付费
```

## 三、核心节奏公式

### 1. "爽-紧-悬"循环

```
爽（主角展示能力/反击/打脸）
  → 紧（反派反扑/新危机出现/更大威胁）
    → 悬（结尾留钩子，观众想看下一集）
```

**典型案例《南瓜》第4集：**
- 爽：南瓜藤抽王彪、苦瓜塞嘴、王彪拉肚子
- 紧：王彪带人来砸菜园+浇柴油
- 悬：沈清辞开玛莎拉蒂出现，"住手！"

### 2. "3秒一反转，10秒一打脸"

短剧容忍度极低，**不能超过30秒没有冲突**。

**检验标准：任何一个超过5行没有对话/动作的纯描述段落，都是节奏拖沓。**

### 3. "欲扬先抑"三步法（出现频率最高）

```
Step 1: 主角被极度羞辱（被骂傻子/被扇耳光/被看不起/被捉奸）
Step 2: 旁观者嘲笑/嘲讽/补刀（村民/配角/反派）
Step 3: 主角反击，围观者震惊/打脸
```

**每个剧本的前10集必须出现至少3次。**

### 4. 冲突密度

| 指标 | 要求 |
|------|------|
| 每集冲突次数 | ≥2 次 |
| 每集打脸次数 | ≥1 次 |
| 每3集身份/能力升级 | ≥1 次 |
| 每5集新人物/新线索 | ≥1 次 |
| 每10集大高潮 | 1 次 |

## 五、写作技法（13部验证）

### 1. 视觉化描写优先

```
❌ 陈石头很痛苦
✅ 陈石头抱着脑袋，痛苦地跪在地上
✅ 鲜血从额角流下，滴入焦黑的土地

❌ 陆云萝很得意
✅ 陆云萝（笑眯眯）：好喝吧，我先走了
✅ 陆云萝OS：这成色，赚大发了啊！就喜欢和地主家的傻儿子做生意！
```

### 2. "OS 补动机"而非"对话补动机"

内心独白用于解释角色**为什么这么做**，而不是描述正在发生什么：

```
✅ 林秀云OS：这些年为了给石头看脑袋，家里入不敷出……
✅ 乔念OS：与其像个皮球一样被踢来踢去，不如抓住眼前的救命稻草！
✅ 叶慕兮OS：前世他就是借着这场踏青宴，毁了我的名声……这一世，我定要让他自食恶果。
```

### 3. 低成本趣味点 = 拟人化道具

南瓜的"颜文字植物"是短剧中最便宜的笑点来源：

```
南瓜VO：坏蛋敢欺负主人！吃小爷一鞭！(๑•̀ㅂ•́)و✧
黄瓜VO：主人好，我是能美容养颜的白玉黄瓜 (^▽^) ！
```
**低成本+高记忆点=短剧爆款标配**

### 4. 场景切换公式

```
内景 → 外景（或反向）
白天 → 夜晚
弱势场景 → 强势场景
```
每2-3集必须有一次明显的环境切换，防止视觉疲劳。

### 5. 台词密度规范

- **短句为主**：绝大多数台词 < 15 字
- **OS/VO 穿插**：每集约 5-8 句内心独白
- **沉默/动作描写**：每集 3-5 处（用于情感/紧张时刻）
- **禁止长段独白**：超过3句的连续独白=拖沓

## 九、付费点设计（一卡卡点）

### 第10集必须满足的条件

1. **大高潮**：前9集积累的矛盾在此爆发
2. **新悬念**：揭示更大的威胁/更强的敌人
3. **身份变化**：主角获得新身份/新能力/新关系
4. **情感高潮**：男女主感情推进或重大转折

### 经典卡点案例

```
南瓜第10集：菜地被烧+沈清辞被偷袭 → 悬念：主角能否救出两人？
海王第3集：接引使被杀+红衣船灵转头 → 悬念：红衣是谁？
侍卫第10集：被弯刀刺穿胸膛 → 悬念：能否觉醒刀法宗师？
嫡女第10集：桃花别院对质+皇后参奏 → 悬念：能否全身而退？
```

## 十一、新增规律（下山祸害/牛马编剧/铁拳镇山河 补充）

### 1. 开局"VO/旁白定调"写法（3/3 新增剧本验证）

这三个剧本开头都用**画外音+快速闪切画面**建立世界观，替代传统的角色对话开场：

```
下山祸害：
  VO: 昆仑山上的修炼异象 → 师傅一鞋拍飞 → 踢下山

牛马编剧：
  老板摔剧本 → 女主砸文件 → 摔倒撞桌角 → 穿越

铁拳镇山河：
  VO: "渊城，十里洋场烟花地！" → 舞厅 → 码头厮杀 → 全家被杀
```

**公式：`VO定调 → 快速闪切3个画面（每画面2-3行）→ 主角陷入极端处境`**

### 2. "信息差打脸"的升级写法（下山祸害验证）

下山祸害展示了**"让敌人自己吃自己下的药"**的经典模式：

```
齐天宇下药 → 狗调包酒杯 → 齐天宇喝下自己的药 → 被狗扑倒
```

**规则：反派下的毒/设的陷阱，最终必须原封不动回到他自己身上。**

### 3. 恐怖/悬疑副本的"规则反着读"模式（牛马编剧验证）

牛马编剧展示了一种新型金手指：**熟知副本规则的反面**。

```
表面规则：拉满100圈兑换免死金牌 → 实际：第99圈会死
表面规则：末位淘汰最低绩效 → 实际：淘汰最高绩效
表面规则：工作才能活 → 实际：摸鱼才安全
```

**公式：`表面规则 → 观众以为是真的 → 主角知道是反的 → 违反规则反而安全`**

### 4. 民国/武侠题材的"比武擂台"结构（铁拳镇山河验证）

铁拳镇山河展示了一种**多势力博弈+擂台比武**的经典结构：

```
四大帮会聚宴 → 反派施压 → 女主泼茶反击 → 提议比武 → 
第一场：我方胜 → 第二场：敌方派出隐藏高手 → 我方认输/主角上场
```

**规则：每场比武必须有"己方以为稳了→被反杀→主角出手"的三段结构。**

### 5. "宠物/动物配角"的搞笑功能（下山祸害验证）

豆豆（京巴狗）是下山祸害中最受欢迎的角色：
- 说话（龙气开智）
- 撒尿写"2b"字
- 调包酒杯
- 咬反派大腿

**公式：`动物配角 + 人性行为 + 关键时刻立功 = 低成本爆笑点`**

### 6. "后宫/多女主"的节奏控制（下山祸害验证）

下山祸害展示了如何处理多个女性角色而不让剧情散乱：

```
大师姐 = 靠山+资本（每次出场带资源/解决大问题）
二师姐 = 搞笑+调戏（调节气氛）
小师妹 = 武力担当（关键时刻救场）
四老婆 = 剧情冲突核心（被下蛊、被陷害）
五老婆 = 情感支线（父亲生病需要救治）
```

**规则：每个女性角色必须有独立功能，不能只是"漂亮+爱主角"。**

### 7. 新增反模式

| 反模式 | 案例 | 问题 |
|--------|------|------|
| 师父踢下山直接开局 | 下山祸害 | 太老套，需有新意 |
| 龙气让所有女角色发情 | 下山祸害 | 低俗擦边，容易被审核 |
| 恐怖副本纯靠主角光环 | 牛马编剧 | 如果主角没信息差就太假 |
| 民国戏全篇方言 | 铁拳 | 影响传播 |
| 比武全是慢动作 | 铁拳 | 短剧不能慢 |

## 十二、常见错误（反模式）

### ❌ 绝对不要做的事

1. **第1集没有金手指展示** — 13/13 精品全部在第1集展示了
2. **连续两集用同一种钩子** — 观众会审美疲劳
3. **反派太复杂/太多内心戏** — 短剧反派要脸谱化
4. **超过30秒无冲突** — 观众会划走
5. **以平淡场景结尾** — 每集最后必须留悬念
6. **主角被动等待救援** — 即使是甜宠，女主也要主动
7. **长段描述超过5行** — 节奏拖沓
8. **台词超过15字** — 短句为王
9. **一卡内没有身份升级** — 观众看不到成长
10. **全60集没有马甲/身份反转** — 缺少大爆点
